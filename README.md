# ___SUR_TAAL___

🎵 SUR TAAL STUDIO

A modern, responsive music studio website built using HTML, CSS, and JavaScript.

🔗 Live Website:
https://subhrajitnaskar.github.io/___SUR_TAA____/

---

📖 About

Sur Taal Studio is an affordable and professional rehearsal & recording studio designed for musicians, bands, singers, and content creators.

The website showcases the studio's facilities, equipment, pricing, booking information, gallery, and contact details in a modern and user-friendly design.

---

✨ Features

- 🎨 Modern Responsive Design
- 📱 Mobile-Friendly Layout
- 🖼️ Interactive Equipment Gallery
- 🎵 Studio Equipment Showcase
- 📸 Studio Photo Gallery
- 💰 Pricing & Add-On Equipment Details
- 📅 Booking Form with WhatsApp Integration
- ⭐ Customer Reviews Section
- 📍 Google Maps Location Link
- 📞 Contact Information
- 📷 Instagram Integration
- 💬 Floating WhatsApp Contact Button
- 🚀 Smooth Scrolling Navigation
- ✨ Scroll Reveal Animations
